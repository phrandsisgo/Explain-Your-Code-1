Kleiner Wissenstest | Was ist der HTML DOM?

ein Dom (steht für Document Object Model) ist im prinzip die Schnittstelle zwischen HTML Elemente und js dass sich sichtbar (kann auch unsichtbar sein) auf deiner webseite befindet. Alle HTML Elemente werden dabei zu einem Objekt.
